# إعداد فرع Expo لتشغيل التطبيق على الهواتف

تم إنشاء فرع مخصص لتجربة التطبيق عبر Expo على أجهزة Android و iOS. هذا الدليل يوضح كيفية التبديل من نسخة الويب إلى نسخة الهواتف.

## المتطلبات المسبقة
- Node.js 18 أو أحدث
- pnpm أو npm
- تطبيق **Expo Go** مثبت على الهاتف
- Android Studio أو Xcode إذا كنت تريد البناء على المحاكي أو إنشاء ملفات APK/IPA

## خطوات العمل على فرع Expo
1. **التبديل إلى فرع الهواتف**
   ```bash
   git checkout expo-mobile
   ```
   استخدم اسم الفرع الذي تم إنشاؤه للموبايل إذا اختلف عن المثال.

2. **تثبيت التبعيات**
   ```bash
   pnpm install
   ```

3. **تشغيل التطبيق على الهاتف**
   - لتشغيله على جهاز حقيقي عبر الشبكة المحلية:
     ```bash
     npx expo start --lan
     ```
   - لتشغيله عبر نفق في حال اختلاف الشبكات:
     ```bash
     npx expo start --tunnel
     ```

4. **اختيار المنصة**
   بعد تشغيل الأمر السابق سيظهر QR Code في الطرفية/لوحة التحكم الخاصة بـ Expo:
   - امسح الـ QR عبر تطبيق **Expo Go** على الهاتف.
   - أو اضغط `a` لتشغيله على محاكي Android، و `i` لتشغيله على محاكي iOS (على macOS).

5. **بناء تطبيق مخصص (اختياري)**
   إذا أردت بناء نسخة تطبيق مستقلة بدلاً من Expo Go:
   ```bash
   npx expo run:android   # إنشاء APK/تشغيله على جهاز Android متصل
   npx expo run:ios        # يتطلب Xcode و macOS
   ```

## ملاحظات مهمة

- تأكد من تحديث متغيرات البيئة الخاصة بالـ API والتخزين لتعمل على الهاتف.
